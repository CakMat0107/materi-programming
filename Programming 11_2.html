1. Panduan Setup Lingkungan Kerja (Step-by-Step)

Langkah 01

Instalasi Python 3

Unduh installer resmi dari python.org. Pilih versi Python 3 terbaru.

Wajib Dicentang (Windows)!

Saat jendela installer pertama muncul, centang opsi "Add python.exe to PATH" sebelum mengklik Install Now.

Langkah 02

Instalasi Visual Studio Code

Unduh installer dari code.visualstudio.com dan selesaikan proses instalasi hingga selesai.

  • Gunakan pengaturan standar (*default*) installer.
  • Gunakan VS Code sebagai Code Editor utama praktikum.
Langkah 03

Pasang Ekstensi Python

Buka VS Code, pilih ikon **Extensions** di sidebar kiri (atau tekan Ctrl+Shift+X).

1. Cari: "Python"
2. Pilih ekstensi resmi terbitan Microsoft
3. Klik tombol Install
Langkah 04

Verifikasi Instalasi di Terminal

Buka terminal terintegrasi di VS Code dengan menekan shortcut Ctrl + ` (backtick).

$ python --version
Python 3.11.x (Berhasil!)

2. Anatomi Skrip Python Pertama (hello.py)

hello.py Python 3 
# Menampilkan pesan salam ke layar terminal

"Hello, World!"

Diawali dengan tanda pagar #. Dibaca oleh manusia, diabaikan oleh komputer.

4. Troubleshooting Cepat (Solusi Error Umum)

Pesan Kendala / Error Penyebab Umum Solusi Penanganan
'python' is not recognized... Lupa mencentang Add to PATH saat instalasi Python di Windows. Jalankan ulang file installer Python, pilih Modify, lalu centang Add Python to environment variables.
SyntaxError: invalid syntax Lupa tanda kurung atau tanda petik saat menuliskan fungsi print(). Pastikan penulisan lengkap: print("teks di sini").
Tombol Run tidak merespons Ekstensi Python di VS Code belum aktif atau terpasang sempurna. Buka menu Extensions (Ctrl+Shift+X), pastikan ekstensi Python (Microsoft) sudah terpasang.
