1. Apa itu Computer Science?

Mitos Populer

Belajar CS berarti sekadar belajar mengetik cepat, memperbaiki laptop rusak, menginstal Windows, atau memakai Word/Excel.

Fakta Sebenarnya

Computer Science adalah seni memecahkan masalah (Problem Solving) secara sistematis dengan instruksi yang presisi kepada komputer.

# Algoritma Cek Kelulusan dalam Bahasa Python

def cek_kelulusan():
    # 1. Input Nilai dari Pengguna
    nilai = float(input("Masukkan Nilai Siswa: "))
    
    # 2. Pengondisian / Decision (Apakah Nilai >= 75)
    if nilai >= 75:
        print("✅ Output: Selamat, Anda LULUS!")
    else:
        print("❌ Output: Mohon maaf, Anda harus REMIDI!")

# Jalankan Fungsi

Jika Total Harta ≥ Nisab (85gr Emas), maka Wajib Zakat 2.5%, jika tidak maka Bebas Zakat.

ganjil_genap.py 
# Solusi Opsi A: Menentukan Bilangan Ganjil atau Genap

angka = int(input("Masukkan sebuah angka bulat: "))

# Memeriksa sisa bagi dengan angka 2
if angka % 2 == 0:
    print(f"Angka {angka} adalah Bilangan GENAP")
else:
    print(f"Angka {angka} adalah Bilangan GANJIL")
kalkulator_zakat.py 
# Solusi Opsi B: Kalkulator Zakat Maal Sederhana

HARGA_EMAS_PER_GRAM = 1100000  # Asumsi Rp 1.100.000 / gram
NISAB_HARTA = 85 * HARGA_EMAS_PER_GRAM  # Setara 85 gram emas

total_harta = float(input("Masukkan Total Harta Anda (Rp): "))

if total_harta >= NISAB_HARTA:
    zakat = total_harta * 0.025  # Zakat 2.5%
    print(f"Wajib Zakat! Jumlah Zakat yang Harus Dibayar: Rp {zakat:,.0f}")
else:
    print("Bebas Zakat (Harta belum mencapai batas Nisab 85 gr Emas).")
