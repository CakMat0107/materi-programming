Low-Fidelity (Wireframe) Phase

Fase konsentrasi pada **tata letak (layout)**, hirarki informasi, dan navigasi tanpa terdistraksi oleh warna, foto, atau gaya visual. Menggunakan elemen abu-abu dan teks tempat sementara (*placeholder*).

Aturan Utama pada Ukuran 390 × 844 px:
  • Safe Area Margin: Berikan padding samping minimal 16px - 20px agar konten tidak menempel ke pinggir layar.
  • Touch Target Size: Tombol utama minimal setinggi 44px - 48px agar mudah ditekan oleh jempol.
  • Grid System: Gunakan 4-Column Layout Grid di Figma dengan margin 16px dan gutter 12px.

Langkah Praktikum Design Mobile App

Tahapan merancang aplikasi seluler di Figma dari nol hingga siap di-prototype.

01

Menyiapkan Canvas & Grid

Tekan shortcut F, pilih preset iPhone 13 / 14 (390×844). Tambahkan Layout Grid 4 Kolom dengan margin 16px.

02

Penerapan Auto Layout

Seleksi elemen lalu tekan Shift + A untuk mengaktifkan Auto Layout agar komponen bersifat responsif secara otomatis.

03

Design System & Component

Buat Reusable Component untuk Tombol, Card, dan Navigation Bar menggunakan Ctrl + Alt + K.
