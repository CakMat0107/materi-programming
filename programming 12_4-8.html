Alur Kerja Slicing Proyek (Pertemuan 04 - 08)

Tahapan bertahap memindahkan desain dari Figma hingga menjadi web interaktif utuh.

01. INSPECT FIGMA

Analisis Layout

Cek ukuran padding, margin, warna (HEX), dan hirarki Auto Layout pada file Figma.

02. HTML STRUCTURE

Markup Semantik

Buat struktur tag HTML5 yang rapi (<header>, <main>, <section>, <button>).

03. TAILWIND STYLING

Styling Utilities

Terapkan kelas Tailwind CSS untuk flexbox, grid, typography, warna, dan responsivitas.

04. JS DOM LOGIC

Interaktivitas

Hubungkan event listener (click/input) untuk mengubah elemen DOM secara dinamis.
