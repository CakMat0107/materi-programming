1. Tipe Data Dasar & Variabel

Tipe Data Kode Python Contoh Nilai Penjelasan Singkat
String (Teks) str "Ahmad", 'Kelas 11' Diapit oleh tanda petik ganda "" atau tunggal ''.
Integer (Angka Bulat) int 16, -5, 2026 Angka bulat tanpa desimal. Bisa bernilai positif/negatif.
Float (Desimal) float 170.5, 85.0 Angka berkoma (menggunakan tanda titik . sebagai pemisah).
Boolean (Logika) bool True, False Hanya bernilai Benar (True) atau Salah (False).

2. Input/Output & Type Casting

Fungsi input() selalu menghasilkan str. Gunakan konversi (*casting*) untuk kalkulasi matematika dan **f-string** untuk mencetak hasil. 

umur = int(input("Umur: "))
print(f"Umur Anda {umur} tahun")

3. Percabangan (IF - ELIF - ELSE)

Evaluasi kondisi menggunakan operator pembanding (>, <, ==, >=) dan operator logika (and, or, not) untuk mengontrol alur program. 

if umur >= 15:
    print("Mukallaf")
else:
    print("Belum Mukallaf")

kalkulator_zakat.py 
NISAB_EMAS = 85.0
total_emas = float(input("Total Emas (g): "))

if total_emas >= NISAB_EMAS:
    zakat = total_emas * 0.025
    print(f"Wajib Zakat: {zakat:.2f} gram")
else:
    sisa = NISAB_EMAS - total_emas
    print(f"Belum Wajib Zakat. Kurang {sisa:.1f} g")

Misi Koding Praktikum:

"Kalkulator Belanja Toko Santri"

Buatlah berkas bernama tugas_kasir.py. Minta pengguna memasukkan **Nama Barang**, **Harga Satuan (Rp)**, dan **Jumlah Barang**. Hitung total bayar dan tampilkan struk belanja menggunakan **f-string**!

tugas_kasir.py 
print("=== TOKO SANTRI COMPUTER SCIENCE ===")

# 1. Input Data Belanja
nama_barang = input("Nama Barang       : ")
harga_satuan = int(input("Harga Satuan (Rp) : "))
jumlah_beli = int(input("Jumlah Dibeli     : "))

# 2. Hitung Total Pembelian
total_bayar = harga_satuan * jumlah_beli

# 3. Tampilkan Struk Pembayaran
print("\n====================================")
print("          STRUK PEMBAYARAN          ")
print("====================================")
print(f"Item       : {nama_barang}")
print(f"Harga @    : Rp {harga_satuan:,}")
print(f"Kuantitas  : {jumlah_beli} pcs")
print("------------------------------------")
print(f"TOTAL BAYAR: Rp {total_bayar:,}")
print("====================================")
